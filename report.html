第二章：核心指标详解

以下详细解释 RSI、MACD、布林带三大核心技术指标的数学原理与实战用法。每个指标底部附有基于最新交易数据的当前值及状态判断。

RSI — 相对强弱指数 (14日)

计算公式：RSI = 100 - 100 / (1 + RS)

RS = N日平均上涨幅度 / N日平均下跌幅度

核心参数：周期 N = 14 日（最常用）

判断逻辑：

  • RSI > 70 → 超买，价格可能回调或下跌
  • RSI < 30 → 超卖，价格可能反弹或上涨
  • RSI 在 30-70 → 中性震荡，无明显信号
  • RSI 顶背离（价格新高 RSI 未新高）→ 上升动能衰竭，看跌
  • RSI 底背离（价格新低 RSI 未新低）→ 下跌动能衰竭，看涨

计算公式：

DIF = EMA(close, 12) - EMA(close, 26)
DEA = EMA(DIF, 9)
MACD 柱 = 2 x (DIF - DEA)

布林带 — Bollinger Bands (20,2)

计算公式：

中轨 = SMA(close, 20)
上轨 = 中轨 + 2 x StdDev(close, 20)
下轨 = 中轨 - 2 x StdDev(close, 20)

核心参数：周期 20 日 / 标准差倍数 2（覆盖约95%价格波动）

判断逻辑：

  • 价格触及上轨 → 短期超买，可能回调；若持续沿上轨运行则强势
  • 价格触及下轨 → 短期超卖，可能反弹；若持续沿下轨运行则弱势
  • 带宽收窄（缩口） → 波动率降低，往往预示大幅变盘即将到来

  • 中轨方向 → 中轨向上为上升趋势，向下为下降趋势

第四章：扩展指标探索 — ATR（平均真实波幅）

ATR — 平均真实波幅 (Average True Range)

计算方法

TR = max(high - low, |high - pre_close|, |low - pre_close|)
ATR = SMA(TR, 14)（或 EMA(TR, 14)）

第一步：计算真实波幅 TR

取以下三者中的最大值：

  1. 当日最高价 - 当日最低价
  2. |当日最高价 - 前日收盘价|
  3. |当日最低价 - 前日收盘价|

TR 衡量了包含跳空缺口在内的真实价格波动范围。

第二步：平滑计算 ATR

对 TR 序列取 14 日移动平均，得到 ATR(14)。

ATR 的值越大，说明该时期价格波动越剧烈。

ATR 的主要作用

  • 衡量波动率：ATR 不判断方向，只反映波动剧烈程度。高 ATR = 大波动，低 ATR = 横盘整理。
  • 动态止损设置：常用 2x ATR 作为止损距离。例如：买入价 - 2 x ATR = 止损价。这样止损距离随波动率自适应调整，避免被正常波动扫出。
  • 仓位管理：ATR 越大表示风险越大，应减小仓位；ATR 越小表示市场平稳，可适当加大仓位。
  • 突破确认：价格突破关键位且 ATR 同步放大，说明突破有波动率配合，假突破概率更低。
  • 趋势强度辅助：趋势行情中 ATR 通常较高；震荡行情中 ATR 通常较低。
